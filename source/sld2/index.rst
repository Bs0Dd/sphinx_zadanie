
Formy komercjalizacji
=======================

.. image:: formy.png

Jakie są formy komercjalizacji?

+--------------------------------+-------------------------------------+
|  Komercjalizacja bezpośrednia  |      Komercjalizacja pośrednia      |
+================================+=====================================+
| *Osobiste zaangażowanie się    | *Twórca innowacji dokonuje jej      |
| twórcy w proces                | komercjalizacji w formie sprzedaży  |
| komercjalizacji*               | prawa własności do innowacji        |
|                                | podmiotowi, który chce tę           |
|                                | innowację wprowadzić na rynek.*     |
+--------------------------------+-------------------------------------+
| * Spin-off - utworzenie nowej  | * Licencjonowanie patentu lub       | 
|   firmy przez naukowców lub    |   technologii - udzielenie innym    |
|   instytucję badawczą w celu   |   firmom prawa do korzystania z     |
|   komercjalizacji              |   opatentowanego wynalazku lub      |
|                                |   technologii w zamian za opłatę    |
+--------------------------------+-------------------------------------+
| * Sprzedaż know-how -          | * Licencja na oprogramowanie lub    |     
|   jednorazowa transakcja       |   know-how - sprzedaż lub           | 
|   polegająca na sprzedaży      |   udostępnienie prawa do używania   | 
|   wiedzy technologicznej       |   oprogramowania, procedur lub      | 
|   innej firmie.                |   specjalistycznej wiedzy           | 
|                                |   technicznej, bez przenoszenia     | 
|                                |   własności.                        | 
+--------------------------------+-------------------------------------+  
| * Wdrożenie własnego wynalazku | * Współpraca z przemysłem -         | 
|   - samodzielne wprowadzenie   |   partnerstwo naukowców z firmami w |
|   wynalazku na rynek przez     |   celu wspólnego prowadzenia badań, |
|   autora                       |   lub wdrożenia innowacji.          |
+--------------------------------+-------------------------------------+
